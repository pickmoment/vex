# VEX — Visual EXplorer
> "파일을 보는 새로운 방식 — 터미널 안의 뷰어"

---

## 1. 컨셉 & 포지셔닝

| 항목 | 내용 |
|------|------|
| 목표 | yazi보다 직관적이고, 파일 **보기(View)** 경험에 집중한 TUI 파일 매니저 |
| 구현 언어 | **Rust** (ratatui + tokio 기반) |
| 플랫폼 | Linux / macOS / Windows (크로스플랫폼) |
| 핵심 철학 | **"열기 전에 이미 다 보인다"** — 미리보기가 1등 시민 |

### yazi와의 차별점

| | yazi | VEX |
|---|---|---|
| 미리보기 방식 | 우측 패널 (고정) | **패널 / 전체화면 / 플로팅** 3모드 |
| 마크다운 | 원문 텍스트 표시 | **렌더링된 HTML 스타일 TUI** |
| 이미지 | 터미널 프로토콜 의존 | 자체 픽셀 렌더러 + 프로토콜 폴백 |
| 학습 곡선 | Vim 키맵 중심, 가파름 | **UI 힌트 상시 표시**, 마우스 완전 지원 |
| 설정 | TOML 직접 편집 | **TUI 내 설정 화면** 제공 |

## 3. 핵심 기능 명세

### 3-1. 마크다운 렌더링 프리뷰
- **pulldown-cmark** 파싱 → ratatui 위젯으로 렌더링
- 지원 요소: 제목(H1~H6), 굵기/기울임, 코드블록(신택스 하이라이팅), 테이블, 인용구, 체크리스트, 수평선
- `[Raw]` / `[Render]` 토글로 원문 ↔ 렌더링 전환
- 코드 블록: **syntect** 기반 170+ 언어 하이라이팅

### 3-2. 이미지 / PDF 미리보기
- **이미지**: Kitty / iTerm2 / Sixel 프로토콜 순차 시도 → 미지원 시 Braille 픽셀 폴백
- **PDF**: `pdfium-render` 바인딩으로 페이지 래스터라이즈 → 터미널 픽셀 출력
- Windows: Windows Terminal / WezTerm에서 Sixel 지원
- 지원 포맷: PNG, JPEG, GIF(애니), WEBP, SVG, PDF

### 3-3. 전체화면 파일 뷰어
- `Space` 또는 더블클릭으로 즉시 전환
- 파일 타입별 전용 뷰어:
  - 텍스트/코드 → 신택스 하이라이팅 뷰어
  - 마크다운 → 렌더링 뷰어
  - 이미지 → 최대 해상도 픽셀 뷰어
  - PDF → 페이지 넘김 뷰어 (← →)
  - CSV → 정렬 가능한 테이블 뷰어
  - ZIP/tar → 압축 내부 트리 뷰어 (추출 없이)
- `[이전 파일]` / `[다음 파일]` 로 폴더 내 탐색 유지

### 3-4. 텍스트 편집기 연동
- `e` 키 → `$EDITOR` / `%EDITOR%` 실행 (vim, nvim, vscode, notepad++ 등)
- VEX 내장 **미니 에디터** (간단 메모 수준): 별도 편집기 없어도 즉시 편집 가능
- 저장 후 자동으로 VEX 미리보기 갱신

## 4. UX 원칙

### 직관성 우선 설계
1. **상시 힌트 바**: 하단에 현재 컨텍스트에서 사용 가능한 단축키 항상 표시
2. **마우스 완전 지원**: 클릭, 더블클릭, 우클릭 메뉴, 스크롤, 드래그 선택 전부 지원
3. **명령어 팔레트**: `Ctrl+P` → 기능명 검색으로 실행 (단축키 몰라도 OK)
4. **TUI 설정 화면**: `Ctrl+,` → 설정을 직접 파일 편집 없이 UI로 변경
5. **온보딩 투어**: 첫 실행 시 인터랙티브 튜토리얼 (스킵 가능)

### 키맵 철학
- 기본값은 **직관적인 단일 키** (vim 스타일 강제 없음)
- `hjkl` 지원하되, **화살표키가 동등하게 동작**
- `?` 키로 언제든 단축키 전체 목록 오버레이

## 5. 기술 스택

| 레이어 | 크레이트 / 기술 |
|--------|----------------|
| TUI 프레임워크 | `ratatui` 0.28+ |
| 비동기 런타임 | `tokio` |
| 파일시스템 감시 | `notify` (실시간 변경 감지) |
| 마크다운 파싱 | `pulldown-cmark` |
| 신택스 하이라이팅 | `syntect` |
| 이미지 렌더링 | `ratatui-image` + `image` crate |
| PDF 렌더링 | `pdfium-render` |
| 퍼지 검색 | `nucleo` (fzf 대체, Rust 네이티브) |
| 설정 포맷 | TOML (`serde` + `config`) |
| 크로스플랫폼 경로 | `dirs` crate |

## 6. 뷰어 모드 전환 플로우

```
파일 목록
    │
    ├─ 커서 이동 ──────────────→ 우측 패널 자동 미리보기 (실시간)
    │
    ├─ Space / 더블클릭 ───────→ 전체화면 뷰어
    │                               ├─ q / Esc → 파일 목록 복귀
    │                               ├─ e       → 외부 편집기
    │                               └─ ← →     → 이전/다음 파일
    │
    └─ Enter ──────────────────→ 기본 앱으로 열기 (OS 연동)
```

├── src/
│   ├── main.rs

│       └── mod.rs        # 설정 관리
